Deskripsi Singkat

WebGIS atau dikenal dengan Website Geogrhapic Information System merupakan sebuah aplikasi perkembangan dari GIS dalam bentuk website yang dapat diakses oleh para pengguna dengan hanya menggunakan internet. Pembuatan website tersebut dilakukan dengan menggunakan bahasa HMTL, CSS, dan JavaScript untuk menghasilkan tampilan yang interaktif. Materi terkait WebGIS ini merupakan salah satu hal penting yang harus dimiliki oleh seseorang sarjana lulusan teknik geodesi ataupun lainnya yang ingin fokus pada bidang Web Developer. Salah satu contoh website yang dibentuk dengan menggunakan bahasa pemrograman tersebut yaitu website Teknik Geodesi UGM yang dapat diakses pada link berikut ini:Link Website Teknik Geodesi UGM. Pengenalan HTML, CSS, dan JavaScript

HTML atau Hypertext Transfer Markup Language merupakan sebuah bahasa pemrograman yang digunakan dalam pembuatan website dengan mengisikan elemen-elemen html didalamnya, yang ditandai dengan adanya tag pembuka dan tag penutup. HTML digunakan untuk menyusun kerangka dasar webiste, termasuk yang hendak ditampilkan pada sebuah halaman web. CSS merupakan bahasa yang digunakan untuk mengatur tampilan dari website yang dibentuk. Tampilan ini terkait dengan color, size, font, dan lainnya. Sedangkan JavaScript merupakan bahasa pemrograman yang digunakan untuk menjalankan suatu fungsi atau perintah terkait dengan aksi pada website tersebut.

Pengenalan Boostrap

Boostrap merupakan salah satu komponen dari website untuk menghasilkan tampilan yang interaktif dan mudah dipahami oleh para pengguna. Pada bagian ini, penggunaan boostrap dilakukan pada saat melakukan pembentukan sebuah CV, yang mana berkaitan dengan navbar, button, dan lainnya. Pelaksanaan kegiatan praktikum mengenai hal tersebut dilakukan dengan menggunakan template yang telah tersedia pada web boostrap dan pengguna dapat melakukan pengeditan untuk menyesuaikan tampilan yang diinginkan.

Pengenalan LeafletJS

Leaflet merupakan sekumpulan library yang berbasisi JavaScript untuk menampilkan peta atau informasi secara interaktif, sehingga dapat dengan mudah dipahami oleh para pengguna.
